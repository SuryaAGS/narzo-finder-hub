# Village Finder 📍

**Digitizing Rural Retail Connectivity**

Village Finder is a Progressive Web App (PWA) designed to bridge the digital gap in rural India. It allows local shopkeepers to list their inventory so villagers can find exactly what they need without traveling long distances.

## 🚀 Features
- **Real-time Inventory:** Shopkeepers can update stock instantly.
- **Local Search:** Users can find specific items in nearby kirana stores.
- **PWA Ready:** Works offline and can be installed on any mobile device.

## 🛠️ Built With
- **Frontend:** React, Tailwind CSS
- **Tools:** Lovable AI, GitHub
